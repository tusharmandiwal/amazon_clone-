# Amazon Clone (Frontend)

A **frontend-only clone of Amazon** built using **HTML, CSS, and JavaScript**.  
This project demonstrates the layout and design of an e-commerce homepage.

---

## **Project Overview**

- Home page with **product listings**, **navigation bar**, and **footer**.
- Responsive design for desktop and mobile screens.
- Includes interactive elements using JavaScript (e.g., image sliders, hover effects).
- No backend or database integration — static frontend only.

---

## **Project Structure**

amazon-clone/
├── index.html # Main HTML page
├── styles.css # CSS styles
└── script.js # JavaScript for interactions

---


## **Technologies Used**

- HTML5
- CSS3
- JavaScript (ES6)
